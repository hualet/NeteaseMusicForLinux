##安装
###手动安装
	mkdir build; cd build; qmake ..; make; make install
	sudo gtk-update-icon-cache /usr/share/icons/hicolor/
###deb安装
	即将支持
	
安装完成后，程序就会出现在你的launcher里面了，尽情享用吧 ;)
